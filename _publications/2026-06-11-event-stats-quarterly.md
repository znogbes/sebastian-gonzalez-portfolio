This quarterly report provides **national statistics** on the number of patient safety events recorded to the *Learn from Patient Safety Events (LFPSE)*. I have contributed to the conception, development, and delivery of this quarterly publication in my role as Senior Information Analyst for [patient safety](https://github.com/nhs-england-patient-safety) at NHS England. 

This report is an example of a **fully automated data pipeline** (i.e., from data extraction to production) to handle large volumes of data on a regular basis. The publication uses data from over **800,000 events** and **2,000+ healthcare organisations** in England. This report is produced using `R` and `quarto`, and the code is version-controlled using `Git`. Data visualisations were generated using `ggplot`.

**Latest issue:** [Quarter 4 2025/26 (January, February, March 2026)](https://www.england.nhs.uk/statistics/statistical-work-areas/patient-safety-data/2025-26/q4/)
